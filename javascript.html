Cours JavaScript Complet

JavaScript (JS) est un langage de programmation qui rend les pages web interactives et dynamiques.

1. Introduction à JavaScript

JS s’exécute dans le navigateur et permet de manipuler le HTML, le CSS et le DOM.

Exemple :

console.log("Bonjour le monde !");

Résultat : Affiche dans la console : Bonjour le monde !

16. Programmation Orientée Objet (POO)

La POO permet d’organiser le code en objets avec des propriétés et des méthodes. En JavaScript, on utilise principalement des classes.

Exemple de classe simple :

class Personne {
    constructor(nom, age) {
        this.nom = nom;
        this.age = age;
    }

    saluer() {
        return `Bonjour, je m'appelle ${this.nom} et j'ai ${this.age} ans.`;
    }
}

// Création d'un objet
let eric = new Personne("Eric", 20);
console.log(eric.saluer());

Résultat : Affiche : Bonjour, je m'appelle Eric et j'ai 20 ans.

Héritage (extends)

class Employe extends Personne {
    constructor(nom, age, poste) {
        super(nom, age);
        this.poste = poste;
    }

    travailler() {
        return `${this.nom} travaille en tant que ${this.poste}.`;
    }
}

let anna = new Employe("Anna", 25, "Développeuse");
console.log(anna.saluer());
console.log(anna.travailler());

Résultat :

  • Bonjour, je m'appelle Anna et j'ai 25 ans.
  • Anna travaille en tant que Développeuse.

Méthodes statiques

class Calcul {
    static addition(a, b) {
        return a + b;
    }
}

console.log(Calcul.addition(5, 3)); // 8

Résultat : Affiche 8 (pas besoin de créer un objet)

Encapsulation (propriétés privées)

class Compte {
    #solde = 0;  // propriété privée

    depot(montant) {
        this.#solde += montant;
    }

    getSolde() {
        return this.#solde;
    }
}

let monCompte = new Compte();
monCompte.depot(100);
console.log(monCompte.getSolde()); // 100

Résultat : le solde est accessible uniquement via la méthode getSolde()

Conclusion sur la POO en JavaScript

La POO permet de :

  • Structurer le code en objets
  • Réutiliser du code via l’héritage
  • Encapsuler les données pour plus de sécurité
  • Créer des applications plus modulaires et maintenables

Avec les classes, les objets et l’héritage, JavaScript devient plus puissant pour des projets complexes.
